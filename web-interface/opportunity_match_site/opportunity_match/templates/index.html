Opportunity Match

This project aims to develop a web-based tool to match experts with opportunities such as challenge descriptions, funding programme and project ideas. The tool will allow opportunity-to-expert, expert-to-expert, and opportunity-to-opportunity matching and create automated emails to users based on customised alerts.
